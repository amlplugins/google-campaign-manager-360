# @amlplugins/google-campaign-manager-360

Native `@googleapis/dfareporting` SDK re-exported under the `@amlplugins` namespace.

> Campaign Manager 360 (dfareporting).

## Install

```bash
npm install @amlplugins/google-campaign-manager-360
```

## Use

```ts
import { /* named exports from @googleapis/dfareporting */ } from "@amlplugins/google-campaign-manager-360";
// or
import sdk from "@amlplugins/google-campaign-manager-360";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
